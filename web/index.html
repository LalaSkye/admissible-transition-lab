Layer 1: Calibration

Establish a shared number line, ordering relation, and basic logical connectives. This is the Lincos-like foundation: both systems must agree on 1 < 2 < 3 and basic truth tables before anything topological can be communicated.

Status: Well-established in METI literature (Freudenthal 1960, Devito 1990). Not novel to this proposal.

The Hard Wall — 5 Conditions
  1. Cognitive processes produce sequential, strand-like traces — This is an empirical claim. If a system's cognition is fully parallel with no serialisable bottleneck, braids may not be the right representation.
  2. Those traces interact via identifiable crossings — Not all process interactions look like crossings. Chemical gradients, field effects, or continuous blending may resist braid encoding.
  3. Crossing structure is invariant under surface perturbation — Track Two provides evidence for this in one system (Claude). The claim must hold for arbitrary substrates.
  4. A receiver can reconstruct topology from transmitted symbols — Requires the calibration layer (Layer 1) to actually work. If symbol grounding fails, topology is unrecoverable.
  5. Structural correspondence supports semantic alignment — The speculative leap. Even if topology transfers, there is no proof that "same shape" implies "same meaning."

Formal Theory

The mathematical foundations underlying the governance kernel and braid topology observable layer.

Transition System

A transition system is a triple (S, A, T) where:

  • S — finite set of states
  • A — finite set of actions
  • T : S × A → S — transition function

A transition system says nothing about whether a transition should happen. It only defines what can happen.

Constraint Kernel

A constraint kernel evaluates every transition before execution:

G : S × A → { ALLOW, DENY, ESCALATE, DRIFT }

The admissible action set from state s:

A*(s) = { a ∈ A | G(s, a) = ALLOW }
Drift Detection

Drift occurs when the transition system changes but the kernel does not:

Drift(s) = { a ∈ actions_from(s) | G has no rule for (s, a) }

If Drift(s) is non-empty, the constraint kernel has a gap. The closed-world default (DENY) catches the action, but governance is incomplete by design — not by intention.

Core Claim

"Safety in autonomous systems is achieved by constraining the reachable state space, not by attempting to regulate model outputs."

Model outputs are proposals. State transitions are execution. Governance must operate at the execution boundary — where proposals become reality — not at the proposal layer where they can be ignored.

A policy is a statement about what should happen. A kernel is a function that determines what does happen.
